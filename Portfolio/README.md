# Personal Portfolio Website

A responsive personal portfolio website built using HTML, CSS, and JavaScript.

This project was created to showcase my skills, projects, achievements, and frontend web development knowledge through a modern and interactive portfolio interface.

The website represents my personal developer portfolio where visitors can explore my projects, technical skills, resume, and contact information in a clean and visually appealing format.

---

# 📖 Project Overview

The main objective of this project was to design and develop a professional portfolio website that highlights personal branding and frontend development skills.

The portfolio includes:

* Introduction and profile section
* Skills and technologies section
* Featured project showcase
* Resume integration
* Contact section
* Responsive modern design

The project helped improve understanding of responsive web design, UI structuring, styling techniques, and interactive frontend development.

# ▶️ How To Run The Project

## Option 1 — Open Directly in Browser

1. Download or clone the repository
2. Open the project folder
3. Double-click `index.html`

This will launch the portfolio website in your default browser.

---

## Option 2 — VS Code Live Server (Recommended)

1. Open the project in VS Code
2. Install the **Live Server** extension
3. Right-click on `index.html`
4. Select **Open with Live Server**

This method is recommended for smoother frontend development and testing.

---

## Option 3 — Using Python HTTP Server

Open terminal inside the project folder and run:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

---

## Option 4 — Using Node.js Live Server

Run the following command:

```bash
npx live-server
```

This automatically starts a local development server and opens the project in the browser.
