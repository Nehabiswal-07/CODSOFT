# NeoGlass Calculator

A modern glassmorphism calculator built using HTML, CSS, and JavaScript.

This project was created to practice frontend web development concepts such as responsive layouts, glassmorphism UI design, DOM manipulation, and JavaScript-based calculator logic.

The calculator features a clean and minimal interface inspired by modern neumorphism and glass-effect design trends.

---

# 📖 Project Overview

The main goal of this project was to build a functional calculator with an attractive modern user interface while improving understanding of JavaScript event handling and frontend interactions.

The calculator supports basic arithmetic operations with a visually clean layout and responsive structure.

---

# ▶️ How To Run The Project

## Option 1 — Open Directly in Browser

1. Download or clone the repository
2. Open the project folder
3. Double-click `index.html`

---

## Option 2 — VS Code Live Server (Recommended)

1. Open the project in VS Code
2. Install Live Server extension
3. Right-click `index.html`
4. Select `Open with Live Server`

---

## Option 3 — Using Python HTTP Server

Run this command inside the project folder:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

---

## Option 4 — Using Node.js Live Server

```bash
npx live-server
```
