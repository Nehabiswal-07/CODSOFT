# 🍕 FireCrust Landing Page

A responsive pizza restaurant landing page built using HTML, CSS, and JavaScript.

This project was created to practice frontend web development concepts such as responsive layouts, modern UI styling, structured page design, and interactive website sections.

The website presents a visually engaging landing page for a fictional pizza brand named **FireCrust**, focusing on attractive food presentation and smooth user experience.

---

# 📖 Project Overview

The main objective of this project was to design a clean and modern landing page that looks visually appealing while remaining simple and easy to navigate.

The website includes:

* Hero section with branding
* Product showcase
* Combo and delivery highlights
* Responsive design for different screen sizes
* Interactive frontend structure

The project helped improve understanding of how real landing pages are structured and styled in modern web development.

# ▶️ How To Run The Project

## Option 1 — Open Directly in Browser

1. Download or clone the repository
2. Open the project folder
3. Double-click `index.html`

This will launch the webpage in your default browser.

---

## Option 2 — VS Code Live Server (Recommended)

1. Open the project in VS Code
2. Install the **Live Server** extension
3. Right-click on `index.html`
4. Select **Open with Live Server**

This method is recommended for better development experience.

---

## Option 3 — Using Python HTTP Server

Open terminal inside the project folder and run:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

---

## Option 4 — Using Node.js Live Server

Run the following command:

```bash
npx live-server
```

This automatically starts a local development server and opens the project in the browser.
